-------- STORY OVERVIEW ---------
    Chara wakes in a world of darkness void of any color. Finding a magic shovel they're off to restore color to the world.

-------- CONTROLS ---------
    For movement use [w] [a] [s] [d] or [up] [left] [right] [down] keys
    Go to pause menu with [esc]
    Sprint with [shift]
    Action with [space]

-------- NOTES ---------
Menu has chapter select of each color (all unlocked in advanced)
    Must play from start to see intro
    Must finish final level for conclusion
Tilemap was supposed to be randomly generated, but Unity failed to comply complete random generation
    Random generated loot under each star point 

Graphics created in Corel Painter 2019
